# Titanic Data Analytics Project

A 6-week end-to-end data analytics project analysing the Titanic passenger dataset using Python.

## What this project covers
- Week 1: Data exploration and problem definition
- Week 2: Data cleaning and feature engineering
- Week 3: Data visualization using Matplotlib and Seaborn
- Week 4: Statistical hypothesis testing using SciPy
- Week 5: Machine learning models using Scikit-learn

## Key Finding
Gender, passenger class, and fare were the three strongest predictors of survival — confirmed through EDA, statistical tests, and machine learning feature importance.

## Libraries Used
pandas, numpy, matplotlib, seaborn, scipy, scikit-learn
